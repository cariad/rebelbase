# RebelBase

**RebelBase** is a Python package for working with numbers of different bases.

Full documentation is online at **[rebelbase.dev](https://rebelbase.dev)**.

## Installation

**RebelBase** requires Python 3.9 or later.

```console
pip install rebelbase
```
